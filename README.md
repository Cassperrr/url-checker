# URL Checker

Monorepo for the asynchronous URL checking service test assignment.

## Workspace

- `apps/backend` - NestJS API
- `apps/frontend` - frontend application placeholder

## Commands

```bash
pnpm install
pnpm dev
pnpm lint
pnpm typecheck
pnpm test
pnpm build
```

Frontend scripts will be wired after the frontend app is initialized.
